# MedHub - Reports — Procedures \[GME\]

## Report List

The procedure reports are found under Reports > Procedure Reports. They provide visibility into procedure logging activity, certifications, requirements tracking, verifications, and supervisor data. Unless otherwise noted, these reports are accessible to program administrators and GME Office users.

### Continuity of Care Summary

Identifies continuity of care by tracking repeat clinic activity and the number of procedures performed by a trainee. Output includes individual encounters, number of visits, dates of first and last visit, number of procedures, and number of diagnoses.

### Procedure Certifications Summary

Lists each procedure type with the percentage of certified trainees by PGY level. Shows how many trainees per level are certified and the combined total across all levels. The percent certified is calculated as the number of PGY trainees certified divided by the total number of PGY trainees in the program.

### Procedure Details Report

Displays certification information for both active and graduated trainees, including certifications applied with or without procedure logs. To view graduated trainees, use the "All In-House" filter option under "Residents to Display."

### Procedure Requirements by Resident

Summarizes procedure requirements by trainee. Display options include procedure type requirements or diagnosis requirements.

### Procedure Requirements Summary

Provides a procedure requirements summary across trainees in a program. Filters: program, display type, filter by (active residents, current residents, or requirements progression between dates), requirement completion (all, complete, incomplete), option to include rejected column. The rejected column shows the total number of rejected procedure logs for each trainee per procedure. Output formats: HTML, XLS, Text (Tab, Comma, or Pipe Delimited).

### Procedure Totals by Type

Counts the number of procedures recorded by each procedure type. Filter options include performed, assisted, observed, supervised, read & interpreted, simulated, total number logged, and number certified.

### Procedure Types by Level

Counts the number of procedures logged by trainee PGY level. All program procedures are displayed followed by a total of all custom (write-in) procedures. Includes a Per Resident Average calculated by dividing the total procedures logged by the total number of residents active during the date range.

**Note on write-in procedures:** When trainees log procedures using the write-in/free-text option rather than selecting from the program's defined procedure list, these appear as "Other procedures (typed in)" on summary reports. The individual typed-in procedure names are visible in detail-level reports (Procedure Details Report, Export Procedures/Cases) but are aggregated into a single "Other" line on summary reports like Procedure Types by Level and Procedures Summary.

### Procedure Verifications Report

Summarizes procedure verifications per trainee and per procedure, including the number of times verified, number of times performed, and aggregated verification comments.

**Important:** This report only displays verified procedures logged with the role of "performed." Procedures logged with other roles (assisted, observed, simulated, etc.) do not appear even if verified. To view verifications for non-"performed" roles, use the Export Procedures/Cases function instead.

### Procedure Summary by Level

Provides a summary of logged procedures by level. Display options include procedure types or diagnoses.

### Procedures Summary

Summarizes trainee procedure logging activity by PGY level, showing whether the procedure was performed, assisted, or observed.

### Procedures Summary by Resident

Provides a summary of logged procedures by trainee, including the number of procedures and roles (performed, assisted, observed, supervised).

### Procedures Supervisor Summary

Displays the number of procedures for each faculty member, broken down by category: performed, assisted, observed, and supervised.

### Visit Summary by Rotation

Displays the visit summary by rotation, showing trainee PGY year, rotation, start/end dates, and types of visits.
